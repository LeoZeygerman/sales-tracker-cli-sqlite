# 📦 Sales Tracker CLI

CLI-приложение для управления товарами, написанное на **Python** с использованием **SQLite** и принципов **объектно-ориентированного программирования (ООП)**.

Проект создан в рамках изучения SQL и перехода от хранения данных в JSON к работе с реляционными базами данных.

---

## 🚀 Возможности

- ➕ Добавление нового товара
- 🔍 Поиск товара по ID
- 📋 Просмотр списка всех товаров
- ✏️ Изменение цены товара
- 📦 Изменение количества товара
- 🗑️ Удаление товара

---

## 🛠️ Используемые технологии

- Python 3
- SQLite3
- Object-Oriented Programming (OOP)
